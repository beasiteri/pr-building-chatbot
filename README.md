# Project: Building a Chatbot

This project is based on the following boilerplate as its foundation and extends it with custom features and functionality specific to the application:

**Bun React Express Full-Stack Boilerplate**  
https://github.com/beasiteri/bun-fullstack-boilerplate

## Core Stack
- **[Bun](https://bun.com)** – fast JavaScript runtime 
- **[Express](https://expressjs.com/)** – backend API
- **[React](https://react.dev)** + **[Vite](https://vite.dev/)** + **[TypeScript](https://www.typescriptlang.org/)** – frontend
- **[TailwindCSS](https://tailwindcss.com/)** – styling
- **[shadcn/ui](https://ui.shadcn.com/)** – UI components
- **[Prettier](https://prettier.io/)** – code formatting
- **[Husky](https://typicode.github.io/husky/)** + **[lint-staged](https://github.com/lint-staged/lint-staged)** – pre-commit formatting
